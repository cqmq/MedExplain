# MedExplain Test Materials

Synthetic test pack for manual MedExplain QA. These files are fictional and
must not be treated as real medical records.

## Folder Contents

- `text/`: copy-paste report examples for the text tab.
- `pdf/`: uploadable PDF reports.
- `images/`: uploadable PNG report images.
- `MANUAL_TEST_CASES.md`: step-by-step checklist and expected behavior.

Use these files only with test/demo environments. Do not mix them with real
patient data.
